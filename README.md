# Online Therapist App

A web-based mental wellness application designed to help users reduce stress, manage emotions, and improve mental health through interactive features.

---

##  Features

- Login & Signup system
-  Mood Tracking (Stressed, Sad, Calm, etc.)
-  Chat Therapist (basic AI responses)
-  Relaxation Music section
-  Daily Motivation Quotes
-  Breathing Exercise for stress relief
-  Responsive and user-friendly UI

---

## Technologies Used

- HTML
- CSS
- JavaScript

---

##  How to Run

1. Download or clone the repository
2. Open `index.html`
3. Run in any browser

##  Live Demo

## Project Objective

To provide a simple and accessible mental wellness platform for students and individuals facing stress and anxiety.
